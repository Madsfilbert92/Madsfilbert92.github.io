Explain in your own words the point of the jitter plot.

The jitter plot is like a dot plot(a placement of datapoints on an axis) except you "jitter" the points in order to distinguish points which have the same value from each other.

When can KDEs be misleading?

KDEs are misleading if you have discrete data where there are no trends in the data. A KDE estimates a continous function, but if your data is discrete this can be misleading. Furthermore it assumes that your data has some kind of trends, but you can not be sure of this before you explore the data.

Janert writes "[CDFs] have less intuitive appeal than histograms or KDEs". What does he mean by that?

Because you no longer look at how much data is a given point, but how much is up to this point, which can be harder to comprehend.

What is a Quantile plot? What is it good for.

It is a CDF but where the x and y axis has switch place. With this you can easily make read as "x% of the data occurs in y or less". With the example from the book "x% of the requests are responded in less than y seconds"

How is a Probablity plot defined? What is it useful for? Have you ever seen one before?

The probability plot is defined by x_i=μ+σΦ^−1(y_i) where μ is the mean, σ is the variance and σΦ^−1 is the inverse of the gaussian cumulative distribution. From this function we can se that if we plot the probability plot the points should approximate a straight line with slope σ and intercept at μ if the data is distributed according to a gaussian distribution. The probability plot is used to confirm whether the data follows a gaussian distribution.

Explain why one has to be careful when using mean, median, and related summary statistics

mean, median and other summary statistics only make sense to interpret if the data is distributed according to an unimodal distribution. Summary statistics may be very misleading if this is not fulfilled. Consider an example where the distribution has two "peaks" then you will probably get a mean that is somewhere between the peaks but you usually interpret the mean as the central tendency in the data but this is not really true if it placed between the two peaks.

I love box plots. When are box plots most useful?

Box plots have a lot of different information in them, such as outliers, median, the inter-quartile range and outliers. It is especially useful when you start out your data exploration. You can easily get a sense if the data is symmetric, and how the data is distributed between the head and the tail and you can spot outliers. Furthermore you can use boxplots to compare distribution between different datasets such that you can find out which one you want to investigate further.

The book doesn't mention violin plots. Are those better or worse than box plots? Why?

When you use a box plot there is some loss of data i.e. you can't see the distribution of the data. The violin plots make up for this by not only having the same information as box plots but also showing the distribution of the data. Thereby you can also get a better of data that does not have a unimodal distribution. However they may not be as good for explanatory data visualisation as not many people know them. You therefore have to make a good description of the plot if you want to use it for communication.
